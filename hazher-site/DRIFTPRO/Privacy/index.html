Personvern — DriftPro (mobilapp)

Sist oppdatert: 17. mai 2026. Gjelder appen DriftPro (Enterprise ERP & HMS) på iOS og Android. Kontakt: kontakt@hazher.no.

Nettsted vs. app: Informasjonskapsler på hazher.no er beskrevet i personvernerklæringen for nettstedet. Denne siden gjelder appen DriftPro.

1. Behandlingsansvar

DriftPro leveres av Hazher. Personvernhenvendelser: kontakt@hazher.no.

2. Hvilke opplysninger appen kan behandle

  • Konto: Navn, e-post, profilbilde, bedrift, rolle, språk og tilgangsinnstillinger per modul.
  • Fravær: Type fravær, datoer, kommentarer, godkjenningsstatus og avdeling.
  • Avvik: Tittel, beskrivelse, kategori, alvorlighetsgrad, bilder, posisjon (valgfritt), anonymitet.
  • HMS: Risikoanalyser, SJA, sikkerhetsrunder, utstyr, kompetanse og tilknyttede dokumenter.
  • DMS: Filer, mapper, tilganger og metadata (f.eks. utløpsdato).
  • Undersøkelser: Svar og deltakelse der du bruker undersøkelsesmodulen.
  • Varsler: Push-token (Firebase) og preferanser dersom aktivert.
  • Teknisk: Logger for drift, sikkerhet og feilsøking.

3. Formål

  • Levere ERP/HMS-funksjoner (fravær, avvik, dokumenter, rapportering).
  • Administrere brukere, roller og godkjenning i bedriften.
  • Sende varsler du har samtykket til.
  • Overholde lovpålagte krav og forebygge misbruk.

4. Tredjeparter

Vi bruker bl.a. Supabase (database og autentisering), Firebase Cloud Messaging (push), og innloggingsleverandører (Apple, Google). Data deles kun i den grad det er nødvendig for disse tjenestene.

5. Dine rettigheter

Du kan be om innsyn, retting eller sletting ved å kontakte kontakt@hazher.no. Du kan klage til Datatilsynet.
